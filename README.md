# snnr-website

The official marketing website for **Snnr Limited** — a New Zealand-based software company specialising in AI, IoT, and cloud-native application development on Microsoft Azure.

## About the site

This is a lightweight, static single-page website that introduces Snnr Limited and the work we do. It covers:

- **Services** — AI-powered applications, IoT solutions, Azure cloud architecture, product development, rapid prototyping, and technical consulting.
- **Portfolio** — products currently in development, including RRM (Restaurant Regulation Management) and MPI compliance forms for cheesemakers and wine producers.
- **About** — background on the company and its founder, Sinan Nar (CEO & CTO, Microsoft Certified Azure Architect).
- **Careers** — opportunities for software engineers and interns.
- **Contact** — how to get in touch about projects, partnerships, or collaboration.

## Tech stack

- Plain HTML and CSS — no frameworks, no build step.
- [Nunito Sans](https://fonts.google.com/specimen/Nunito+Sans) via Google Fonts.

## Project structure

```
src/
├── index.html   # Page content and structure
└── styles.css   # Styling and layout
```

## Running locally

Open `src/index.html` directly in a browser, or serve the `src/` folder with any static file server:

```bash
npx serve src
```
